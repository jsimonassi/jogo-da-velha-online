# Jogo da velha - Online

Este aplicativo está sendo desenvolvido segundo a proposta elencada pelo professor Lauro Kozovits da disciplina de Laboratório de Programação Dispositivos Móveis - UFF.
O objetivo é entender o funcionamento e modelagem de uma aplicação flutter.

### Aplicação base

Como base para o desenvolvimento das ideias, escolhemos o projeto:

     -  https://github.com/iampawan/FlutterTicTacToe 

O mesmo está sendo atualizado para atender os novos requisitos da disciplina.

### Nova abordagem

O projeto base possui uma estrutura totalmente offline, onde o jogador enfrenta um bot(IA). Para o nosso projeto iremos tornar este ambiente multiplayer, permitindo a criação de contas, interação entre amigos, chat e demais funcionalidades que se fizerem necessárias no contexto da aplicação.

### About

This application is being developed according to the proposal listed by professor Lauro Kozovits of the discipline of Programming Laboratory for Mobile Devices - UFF.
The goal is to understand the functioning and modeling of a flutter application.


### Base application

As a basis for the development of ideas, we chose the project:

    -  https://github.com/iampawan/FlutterTicTacToe 

This project is being updated to meet the new requirements of the discipline.

### New approach

The base project has a completely offline structure, where the player faces a bot (AI). For our project we will make this environment multiplayer, allowing the creation of accounts, interaction between friends, chat and other features that may be necessary in the context of the application.
